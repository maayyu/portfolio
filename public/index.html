Java

Javaの基礎構文からオブジェクト指向設計まで幅広く習得

Eclipseを使用した開発環境での演習を通じて、
デバッグ・例外処理・コレクション・パッケージ管理などを経験

できること：

  • クラス／メソッドを用いたロジック設計と再利用性の高い実装
  • 例外処理を考慮した堅牢なプログラム設計
  • ArrayList / HashMap などコレクション操作
  • パッケージ設計とクラス分割による責務の明確化
  • Eclipseでのコンソールアプリケーション開発、デバッグ
  • Gitを利用したソース管理と共同開発

集約・結合・サブクエリなど、実務で必要となるSQL文を網羅的に習得

できること：

  • トランザクション管理（コミット／ロールバック）
  • 権限設定やユーザー管理による安全な運用設計
  • Java（JDBC）からのDB接続・検索・更新処理の実装
  • チーム開発演習におけるGit & GitHubを用いたバージョン管理

Spring

  • フォーム入力値のバリデーション・メッセージ出力の実装
  • Thymeleafを使ったテンプレートレイアウト設計
  • デバッグ・フィルター機能・例外ハンドリングの実践
